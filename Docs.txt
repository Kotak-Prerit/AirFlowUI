### Usage Direction

Using AirflowUI is as simple as:

1. Browse Components ‚Üí Pick the element you want (e.g., Button, Card, Modal, Navbar).
2. Choose Framework ‚Üí Select from HTML, Next.js, Vite.js, Astro, Vue, or Svelte.
3. Copy the Snippet ‚Üí Grab the ready-made code.
4. Paste & Run ‚Üí Drop it into your project. Done.

Want to customize?
- Change Tailwind classes (colors, padding, borders) for quick tweaks.
- Add GSAP or Motion One for animations.
- Integrate Three.js if you want to add interactive 3D elements.

Example: Need a button? Copy it in your framework of choice and make it yours in seconds.
